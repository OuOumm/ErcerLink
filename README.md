# ErcerLink
handsome快速申请友链插件

## 使用教程：

默认使用本地文件，如需使用CDN加速，请填写完整路径，例如：

```https://jsd.onmicrosoft.cn/gh/OuOumm/omo/data/js/ErcerLinks.js```

使用方法，在友联页面添加以下内容：

```<div id="postLink"></div>```

如果使用pjax，请在回调函数内增加以下代码：

```pjax_Link();```
